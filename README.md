# Tome

Tome is a dark-first technical reading tracker built with Vue 3, TypeScript, Vite, Pinia, Vue Router, and Supabase.

## Features
- Supabase auth with profile support
- Book and chapter management
- Chapter progress tracking
- Markdown notes with syntax-highlighted preview
- Flashcard review with spaced repetition
- Pomodoro focus timer
- Supabase SQL schema included in `supabase-schema.sql`

## Setup
```bash
bun install
cp .env.example .env
bun run dev
```

## Build
```bash
bun run build
```
